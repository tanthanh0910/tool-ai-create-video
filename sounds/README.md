# 🔊 Thư mục Tiếng Kêu Động Vật

Đặt file tiếng kêu động vật vào thư mục này.

## 📁 Quy tắc đặt tên

```
sounds/
├── lion.mp3      # Sư tử
├── tiger.mp3     # Hổ  
├── cat.mp3       # Mèo
├── dog.mp3       # Chó
├── cow.mp3       # Bò
├── horse.mp3     # Ngựa
├── elephant.mp3  # Voi
├── wolf.mp3      # Sói
└── ...
```

**Quy tắc:**
- Tên file = tên động vật tiếng Anh (viết thường)
- Thay dấu cách bằng `_` (vd: `polar_bear.mp3`)
- Định dạng: `.mp3`, `.wav`, `.ogg`, `.m4a`
- Độ dài tùy ý (code sẽ tự cắt nếu quá dài)

## 🌐 Nguồn tải miễn phí

1. **Pixabay** - https://pixabay.com/sound-effects/search/animal/
2. **Mixkit** - https://mixkit.co/free-sound-effects/animals/
3. **Freesound** - https://freesound.org (cần đăng ký)
4. **ZapSplat** - https://www.zapsplat.com

## ⚡ Cách hoạt động

- Có file → Phát tiếng kêu sau khi đọc tên
- Không có file → Chỉ đọc tên (không có tiếng kêu)

**Tip:** Chỉ cần tải 10-20 con phổ biến nhất!
